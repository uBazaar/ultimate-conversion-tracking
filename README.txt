Add conversion tracking to your WordPress website, quickly and easily.

== Description ==

Conversion tracking is an essential measurement of website performance, helping to make data-driven decisions. This plugin allows you to setup tracking conversion for four popular networks: LinkedIn, Google Analytics, Facebook, and Twitter.

== Installation ==

1. Upload `ultimate-conversion-tracking.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
